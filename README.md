# Python Backend Practice Projects 

This repository contains all the Python practice projects I built while learning:

* Python Fundamentals
* OOP
* File Handling
* JSON & CSV
* APIs
* Logging
* Regex
* Automation
* Error Handling

## Installation
To install the required packages, run `pip install -r requirements.txt`. If the requirements.txt file does not exist, you can create one by running `pip freeze > requirements.txt` in your terminal. Please note that this project does not have a requirements.txt file, so you will need to create one and add the necessary packages.
